# Terminos de Servicio

Al crear una cuenta o usar EducApp WhiteLabel, el usuario acepta estos terminos, la politica de privacidad y el tratamiento de sus datos personales para las finalidades asociadas a la operacion de la plataforma educativa.

## Uso de la plataforma

El usuario se compromete a utilizar la plataforma de forma licita, responsable y conforme a su perfil asignado. No esta permitido acceder a informacion de terceros, intentar vulnerar controles de seguridad, compartir credenciales o utilizar la plataforma para fines diferentes a los autorizados.

## Cuenta de usuario

Cada usuario es responsable de mantener la confidencialidad de sus credenciales. Las acciones realizadas desde una cuenta autenticada podran ser registradas para fines de seguridad, auditoria y soporte.

## Cursos, evaluaciones y certificados

La plataforma permite acceder a contenidos academicos, presentar evaluaciones, registrar avances y generar certificados cuando se cumplan las condiciones definidas por la institucion o el instructor responsable.

## Pagos y transacciones

Cuando aplique, los pagos se procesaran mediante los proveedores habilitados por la plataforma. Las ordenes, comprobantes y estados de pago se conservaran para control administrativo, financiero y soporte al usuario.

## Propiedad intelectual

Los contenidos, materiales, imagenes, cursos, evaluaciones y recursos publicados en la plataforma pertenecen a sus respectivos titulares. El usuario no puede copiarlos, redistribuirlos o explotarlos sin autorizacion.

## Tratamiento de datos

El usuario autoriza el tratamiento de sus datos personales para autenticacion, gestion academica, administracion de cursos, evaluaciones, certificaciones, pagos, soporte, auditoria, seguridad y comunicaciones relacionadas con el servicio.

## Cambios en los terminos

La plataforma podra actualizar estos terminos cuando sea necesario. En caso de cambios relevantes, se recomienda informar a los usuarios y conservar evidencia de la version aceptada.
