# Politica de Privacidad y Tratamiento de Datos

EducApp WhiteLabel trata datos personales para prestar servicios de formacion, administracion academica, gestion de usuarios, certificaciones, evaluaciones, pagos, soporte y comunicaciones relacionadas con la plataforma.

## Datos que se pueden recolectar

- Datos de identificacion como nombre, correo electronico, documento de identidad, empresa y departamento.
- Datos de autenticacion como contrasena cifrada, sesiones y tokens de acceso.
- Datos academicos como cursos inscritos, progreso, evaluaciones, respuestas, intentos y certificados.
- Datos financieros como ordenes, pagos, comprobantes, transacciones, saldos a favor y devoluciones.
- Datos tecnicos como direccion IP, registros de auditoria, fecha de acceso y actividad dentro del sistema.

## Finalidades del tratamiento

Los datos se utilizan para crear y administrar cuentas, permitir el acceso a cursos, registrar progreso academico, generar certificados, procesar pagos, atender solicitudes de soporte, realizar auditoria de actividad, proteger la seguridad del sistema y cumplir obligaciones legales o contractuales aplicables.

## Seguridad

La plataforma aplica controles tecnicos como autenticacion, autorizacion por roles y permisos, cifrado de contrasenas, proteccion CSRF, control de sesiones y registro de actividad. El acceso a la informacion se limita segun el perfil y los permisos asignados a cada usuario.

## Derechos del titular

El usuario puede solicitar la consulta, actualizacion, correccion o eliminacion de sus datos personales cuando sea legalmente procedente. Tambien puede solicitar informacion sobre el uso dado a sus datos y revocar autorizaciones, salvo que exista una obligacion legal o contractual que requiera conservarlos.

## Pagos

Los pagos pueden ser procesados mediante proveedores externos como MercadoPago. La plataforma no debe almacenar datos sensibles de tarjetas bancarias. Solo conserva la informacion transaccional necesaria para operar ordenes, comprobantes, estados de pago y reportes.

## Conservacion

Los datos se conservaran durante el tiempo necesario para cumplir las finalidades descritas, obligaciones legales, requerimientos contables, soporte academico, trazabilidad, auditoria y seguridad.

## Contacto

Para ejercer derechos sobre datos personales, el usuario debe comunicarse con el responsable de la plataforma o el canal de soporte definido por la institucion que opere EducApp WhiteLabel.
